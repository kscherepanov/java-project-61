## Приложение "Игры разума"
***
Включает в себя 5 математических игр:
* Проверка на чётность
* Калькулятор
* Наибольший общий делитель (НОД)
* Арифметическая прогрессия
* Простое ли число?
***

## Сборка и запуск

```shell
# Только сборка
make build
```

```shell
# Запуск собранного дистрибутива
make run-dist
```
